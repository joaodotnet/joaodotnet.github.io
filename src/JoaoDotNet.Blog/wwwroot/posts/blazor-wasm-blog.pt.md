# Construindo este blog com Blazor

Escolhi Blazor WebAssembly pela identidade .NET e pela liberdade de criar uma experiência interativa sem abandonar C#.

## O que funciona bem

- Componentes reutilizáveis e tipagem forte
- Integracao simples com Markdown e JSON
- Estilo e interações totalmente controlados

## O que deve ser observado

- O bundle inicial é maior do que um site estático tradicional
- SEO é mais simples com geradores, mas aqui não é prioridade

Para um blog pessoal, o equilibrio valeu a pena.
